# HelpDesk Pro – Sistema de Chamados de TI

Projeto desenvolvido como requisito para obtenção do título de Técnico em Informática pela Escola Técnica de Brasília (ETB).

## 📄 Descrição
HelpDesk Pro é uma aplicação web destinada ao gerenciamento centralizado de chamados de suporte técnico em TI. O sistema visa substituir processos manuais (planilhas e e-mails) por uma plataforma estruturada que permite o registro, acompanhamento e histórico completo das solicitações de suporte.

## 🚀 Tecnologias Utilizadas
O projeto foi desenvolvido seguindo a arquitetura MVC (Model-View-Controller) e utilizando as seguintes tecnologias:

* **Backend:** Java (Servlets, JDBC)
* **Frontend:** JSP, HTML5, CSS3
* **Banco de Dados:** MySQL
* **Servidor de Aplicação:** Apache Tomcat
* **Modelagem:** UML (Astah UML)
* **Criptografia:** SHA-256 para senhas

## 🛠 Funcionalidades Principais
* **Autenticação:** Login com controle de perfis (Administrador, Técnico, Usuário).
* **Gestão de Chamados:** Abertura, priorização, atendimento e encerramento de solicitações.
* **Rastreabilidade:** Histórico completo de alterações por chamado.
* **Relatórios:** Geração de relatórios operacionais para auxílio na tomada de decisão.
